JavaScript Basics

Introduction

JavaScript is a dynamic programming language which, when applied to an HTML document, provides dynamic interactivity on websites. It is incredibly versatile, starting with basics such as carousels, image galleries, fluctuating layouts, and responses to button clicks to the ability to create games, animated 2D and 3D graphics, comprehensive database-driven apps, and a whole lot more! JavaScript itself is fairly compact yet very flexible. Developers have written a large variety of tools on top of the core JavaScript language, unlocking a vast amount of extra functionality with minimum effort. These include:

  • Browser Application Programming Interfaces (APIs) â€” APIs built into web browsers, providing functionality like dynamically creating HTML and setting CSS styles, collecting and manipulating a video stream from the user's webcam, or generating 3D graphics and audio samples.
  • Third-party APIs to allow developers to incorporate functionality in their sites from other content providers, such as Twitter or Facebook.
  • Third-party frameworks and libraries you can apply to your HTML to allow you to rapidly build up sites and applications.
    • Variables

    Variables are containers for storing data values. In JS there are essentially four variables, which you should concern yourself with:

  • Integer
  • String
  • Boolean
  • Float

    • Declare your variables using var and give it a name so you can reference it again in your code. To assign an initial value to you variable, simply add "=" together with the value.

    For integers:

    var number = 2;

    For Strings:

    var name = "Ifrah";

    For booleans:

    var married = true;

    For Floats:

    var height = 1.7;
    Functions

    A function is a block of code designed to perform a designated task. It is invoked when something (a user action, a timeout, another function) calls it.

    To declare a function, first give it a name and round brackets () into which you may parse further information (this is optional)

    function myFunction()

    Next add a space and insert a pair of curly brackets, inside which you can insert the code you wish to perform, when the function is called.

    function myFunction() {
    console.log("Hello World!");
    }
    Objects

    Objects are essentially variables, to which you can assign multiple values. These values themselves can take all forms or common variable, including string, integer, boolean, array etc.

    Create a JS object using the "var" keyword and assign it a name.

    var firstPerson

    To add the properties of the object, assign them within a pair of curly brackets, seperating each one with a comma.

    var firstPerson = {
    firstName: "Ifrah",
    height: 1.7,
    age: 23;
    };

    You can also call the individual properties using a dot operator/square brackets.

    console.log(firstPerson.height);
    Comments

    There are two different types of comment you can use in JavaScript, namely:

  • Single-line comment
  • Multiple-line comment

    • To comment out a single line use the following:

    // This is a JavaScript comment;

    To comment out mulitiple lines, open with /* and close with */

    /* This is a
    multi-line
    JavaScript comment */
